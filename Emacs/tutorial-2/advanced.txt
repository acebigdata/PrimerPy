
In mac meta key is option key
C-x 2 <--- split
C-x b <--- switch to a different buffer in a windo
C-x o <--- jump to another buffer
M-x shell <--- Open a terminal
+ <--- to create a directory+
d <--- mark file/folders to delete
x <--- execute delete
g <--- update/refresh to see newly added files
M-% <--- M-x query-replace
M-x replace-string
M-u <--- UPPERCASE
M-l <--- lowercase
M-$ <--- Spell chekc word
